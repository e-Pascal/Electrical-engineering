# Приложение А. Справочник номиналных рядов

Множество номинальных значений компонентов (резисторов, конденсаторов, катушек индуктивности) сведено к так называемым
рядам, которые представляют собой результат стандартизации.

Ниже приведены ряды: E3, Е6, Е12, Е24, Е48, Е96 и Е192, где
число, следующее за буквой Е, обозначает количество значений в ряду для каждого
десятичного интервала.

На практике наиболее часто применяются компоненты из ряда E24 с допуском ±5 % и ряда E96 с допуском ±1 %  

> Например, рассчитанное значение резистора получилось 235,4 кОм и по заданию необъодимо подобрать ближайшее сопротивление из ряда E96. Находим  в столбце E96 таблицы значение 2,37, соответственно результатом подбора из ряда E96 будет резистор с сопротивлением 237 кОм.
